# ESP32 Source Code

This folder contains the ESP32 firmware code for the SafeZone collar devices.

## Purpose

The ESP32 code handles:
- GPS tracking
- Geofence monitoring
- WebSocket communication with the server
- Cow collar hardware integration

## File Structure

Place your ESP32 Arduino/PlatformIO code files here:
- `.ino` files - Arduino sketches
- `.cpp` / `.h` files - C++ source and header files
- `platformio.ini` - PlatformIO configuration (if using PlatformIO)
